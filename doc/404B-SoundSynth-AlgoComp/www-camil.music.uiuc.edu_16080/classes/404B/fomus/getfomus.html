Getting FOMUS:

  1. In OS X, open up Terminal.

  2. Use the “cd” command to change to the directory you want (ex. “cd /the/directory”).

  3. Use the “pwd” command to make sure it's the right directory (ex. “pwd”).

  4. Open up a browser and go to http://common-lisp.net/project/fomus/.

  5. Cut and paste the line that starts with “cvs” into the terminal. Hit enter.

  6. To update it in the future, open Terminal, “cd” to the fomus directory (ex. “cd /the/directory/fomus”) and type “cvs update”. Do this often since I'm still fixing many bugs (please e-mail me if you find any).
